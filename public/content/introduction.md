# Introduction

Welcome to **TERM.DOCS** - a terminal-inspired documentation starter built with Angular 21 Signals and Tailwind CSS.

## What is TERM.DOCS?

TERM.DOCS is a modern documentation engine featuring:

- **Angular 21** with standalone components
- **Signals-based** reactivity for fine-grained updates
- **Terminal aesthetic** with green-on-black design
- **Keyboard-driven** navigation for power users
- **Client-side search** with fuzzy matching
- **SEO-optimized** with meta tags and structured data
- **Production-ready** build pipeline

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
ng serve

# Build for production
ng build
```

## Architecture

The documentation system uses:

- **Content Service** - Loads and parses Markdown files
- **Search Service** - Full-text search with scoring
- **SEO Service** - Manages meta tags and Open Graph
- **Settings Service** - Theme and user preferences
